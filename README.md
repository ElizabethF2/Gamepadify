# Gamepadify

Gamepadify is a utility for remapping and scripting game controllers and other input devices on Linux. It includes a controller-friendly on-screen keyboard.

**NB**: If you experience focus issues with the on-screen keyboard on newer versions of KDE Plasma, you can resolve them by opening the on-screen keyboard, right clicking its title bar, selecting `More Actions` > `Configure Special Application Settings...` and properties then set `Accept focus` to `Force` `No`, `Layer` to `Force` `Overlay` and `Position` to `Remember`.
